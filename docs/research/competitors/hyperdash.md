# Competitor: Hyperdash
- **URL:** https://hyperdash.info/
- **Category:** direct
- **Core Value Prop:** Real-time derivatives analytics — liquidations, funding rates, open interest
- **Tech Stack:** React, Hyperliquid WebSocket API, custom charting
- **Key Features:**
  1. Real-time liquidation heatmap (price levels where positions get liquidated)
  2. Funding rate tracker across all perp pairs
  3. Open interest tracker with historical charts
  4. Top trader leaderboard (PnL rankings)
  5. Position explorer (see what top traders are holding)
- **Known Weaknesses:** Hyperliquid-only (no Binance/Bybit), limited token analytics, no news, no macro context
- **Pricing Model:** Free (Hyperliquid ecosystem tool)
- **User Traction:** 100,000+ monthly active users in Hyperliquid ecosystem
- **Last Major Update:** Position explorer feature (2025)
- **Unique Differentiator:** Best Hyperliquid-native analytics — the "Bloomberg of Hyperliquid"
- **What their users complain about:** Only Hyperliquid data, no cross-exchange comparison, no spot market data
