- **Category:** direct
- **Core Value Prop:** Real-time token discovery and sniper for meme coins and new DEX pairs
- **Tech Stack:** Next.js, Solana RPC, custom DEX scrapers, WebSocket feeds
- **Key Features:**
  1. New pair detection with instant alerts (< 1 minute delay)
  2. Token holder analysis (top holders, concentration)
  3. Smart money tracking per token (which wallets are buying)
  4. Rug detection (honeypot check, liquidity lock verification)
  5. Copy trading (follow whale wallets automatically)

- **Pricing Model:** Free (limited) → $99/mo → $299/mo
- **User Traction:** 500,000+ users, dominant in Solana degen trading
- **Last Major Update:** Base chain support added (2025)
- **Unique Differentiator:** Speed — fastest new pair detection in the market
